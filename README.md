my dotefiles with install script that installs all my preferred programs.
it is not yet ready for public use.

## Installation

Use git to clone the repo.

```bash
git clone https://github.com/project-HOSSAM/Dotefiles.git ~/Dotefiles
```

## Usage

```shell
cd ~/Dotefiles
chmod +x install.sh
./install.sh
```
